# CityEdit — Возможности (Features)

## v1.0.0 — Базовый редактор

### Управление серферами
- Разблокировка/блокировка отдельных серферов
- Установка уровня (1–20) для каждого серфера
- Массовые операции: Unlock All / Lock All / Max Level

### Управление досками
- Разблокировка/блокировка отдельных досок
- Установка уровня (1–20) для каждой доски
- Массовые операции: Unlock All / Lock All / Max Level

### Управление скинами
- Разблокировка/блокировка скинов (привязаны к серферам)
- Массовые операции: Unlock All / Lock All

### Валюта
- Редактирование монет, ключей, токенов и других валют
- Быстрое действие: Max Currency (999999)

### Покупки
- Добавление/удаление записей из истории покупок
- Удаление рекламы (`removeads.01`)
- Сезонный пропуск

### Быстрые действия
- **Unlock Everything** — разблокировка всего в один клик
- **Max Currency** — максимальная валюта
- **Season Pass** — активация сезонного пропуска
- **Remove Ads** — удаление рекламы

### Шифрование
- Автоматическое дешифрование/шифрование файла профиля (AES-128-CTR)
- Сохранение оригинального IV/KEY

## v1.1.0 — Расширенный редактор

### Новые возможности

#### Исправление рекорда серфера
- Отображение и редактирование `highScore` для каждого серфера

#### Уровень и опыт
- Отображение `level` и `xp` в статистике профиля

#### Расширенная база покупок (250+ записей)
- **Категории:** All, Characters, Boards, Skins, Boost Items, Seasonal, Promotions, Infrastructure, System, Other
- **Описания** каждой покупки на русском языке
- **Поиск** по названию и описанию в реальном времени
- Фильтрация по категории

#### Разблокировка бонусных трасс
- 16 бонусных трасс (`districttrial.premiumladder.001` — `016`)
- Кнопка "Bonus Tracks" в QuickActions

#### Выбор активного серфера и доски
- Выпадающие списки в QuickActions
- Изменение применяется мгновенно

#### Редактор флагов
- Новая вкладка FLAGS
- Все профильные флаги с toggle switches
- Реактивное сохранение при переключении

#### Пропуск туториала
- Кнопка "Skip Tutorial" в QuickActions
- Автоматический пропуск при Unlock Everything

#### Улучшенный Unlock Everything
- Теперь также разблокирует бонусные трассы и пропускает туториал
